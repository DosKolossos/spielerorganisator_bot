# Standin-Auswahl repariert

Bei einer unbesetzten Starter-Position war die Option **Ersatz besorgen** bereits als Standardauswahl markiert.
Discord behandelte sie dadurch als aktuellen Zustand statt als auswählbare Aktion.

Die offene Position zeigt nun die auswählbare Option **Sub oder Standin auswählen**.
Nach der Auswahl erscheint die vorhandene Ersatzansicht, in der ein gespeicherter Sub/Standin gewählt
oder über **Neuen Standin anlegen** direkt erstellt und eingesetzt werden kann.
