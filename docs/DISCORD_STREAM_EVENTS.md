# Discord-Events für gestreamte Termine

Discord-Events hängen nicht mehr an der Terminart Prime League.

In der ausgeklappten Adminkarte gibt es den Schalter **Wird gestreamt**. Ein Discord-Event wird erstellt, sobald:

- **Wird gestreamt** aktiviert ist,
- der Status **Fixed** ist und
- eine fixe Start- und Endzeit gesetzt sind.

Das gilt für Prime League, Scrims, Flex und sonstige Termine. Wird der Schalter deaktiviert, der Termin abgesagt oder eine notwendige Zeit entfernt, löscht der Bot das zugehörige Discord-Event.
