# Duplicate Planner Fix

- Übernimmt alte automatische `daily:YYYY-MM-DD`-Einträge in die teambezogene Schlüsselstruktur.
- Löscht doppelte, noch offene Auto-Terminoptionen desselben Teams und Tages.
- Entfernt vorhandene Discord-Karten vor der Datenbankbereinigung, damit keine verwaisten Alt-Nachrichten stehen bleiben.
- Manuelle, fixierte oder abgesagte Termine werden nicht gelöscht.
